(1)  $$\delta(x) = 
\begin{cases}
0,\ если\ x \ne 0 \\
\infty,\ если\ x = 0
\end{cases}$$
(2)
$$\int_{-\infty}^{\infty}\delta(x) dx = 1$$
(3)
$$\int_a^b \delta(x - \xi)dx = 
\begin{cases}
1, \xi \in [a, b] \\
0, \xi \notin [a, b]
\end{cases}$$
(4)
$$\int^b_a f(x) \delta(x - \xi) dx = 
\begin{cases}
f(\xi), a \le \xi \le b \\
0, \xi \notin [a,b]
\end{cases}$$
(5)
$$- \xi(x) = \xi(x)$$
(6)
$$\begin{align}
\int \delta(\alpha x) dx= \int \delta (x) \frac{dy}{|\alpha|} = \frac{1}{|\alpha|} \\
\delta(\alpha x) = \frac{\delta(x)}{|\alpha|}
\end{align}$$
