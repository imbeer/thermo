Теплоемкость бывает
(1) тела

$$c_{тела} = \frac{\delta Q}{dT}$$

(2) удельная (на массу)

$$c = \frac{\delta Q}{mdT}$$

(3) молярная (на количество вещества)

$$\begin{gathered}
c = \frac{\delta Q}{\nu dT} = \frac{dU + \delta A}{
\nu dT} = \\ \\ \frac{\left(\frac{\partial U}{\partial T}\right)_V dT + \left(\frac{\partial U}{\partial V}\right)_T dV + pdV}{\nu dT} = \frac{1}{\nu}\left[\left(\frac{dU}{dT}\right)_V + \left(p+\left(\frac{dU}{dV}\right)_T\right)\frac{dV}{dT}\right]
\end{gathered}$$

$$c_V = / V = const \Rightarrow dV = 0 /= \frac{1}{\nu}\left(\frac{dU}{dT}\right)_V$$

$$c_p = \frac{1}{\nu} \left[ \left(\frac{dU}{dT}\right)_V +  \left(p + \left(\frac{dU}{dV}\right)_T\right) \left(\frac{dV}{dT}\right)_p \right]$$

$$c_p - c_V= \frac{1}{\nu} \left(p + \left(\frac{dU}{dV}\right)_T\right) \left(\frac{dV}{dT}\right)_p$$

В случае идеального газа $\frac{dU(T)}{dV} = 0$, $c_p - c_v = R$
