Вывод уравнения адиабаты.

$$
\delta Q = 0
$$

По перому закону термода:
$$
dU + pdV = 0
$$
Из определения [[06. Теплоемкость|теплоемкости]] 
$$
dU = \nu C_{v} dT
$$


$$
\begin{align}
pV = \nu R T \\
pdV + Vdp = \nu R dT \\
\frac{С_v(pdV + Vdp)}{R} = -pdV \\
\frac{С_v(pdV + Vdp)}{C_p - C_v} = -pdV \\
C_v pdV + C_v Vdp = - C_p pdV + C_v pdV | * \frac{1}{C_v} \\
pdV + Vdp = -\frac{C_p}{C_v}pdV + pdV \\
\frac{dp}{p} = - \gamma \frac{dV}{V}, \ \  \gamma = \frac{C_p}{C_v} \ | взять \ интеграл \\
\ln p = -\gamma \ln V + const \\
\ln(pV^{\gamma}) = const \\ 
pV^{\gamma} = const
\end{align}
$$
Уравнение адиабаты:
$$
pV^{\gamma} = const, \ \ \ \ \gamma = \frac{C_p}{C_v}
$$
